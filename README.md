## covdet

C++ API for covdet of VLFeat

## Dependencies

- OpenCV, 3.x or 2.x
- VLFeat, [VLFeat 0.9.20](http://www.vlfeat.org/)
- Armadillo, [armadillo-8.x](http://arma.sourceforge.net/download.html)
